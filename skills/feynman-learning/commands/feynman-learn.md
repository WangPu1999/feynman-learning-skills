# feynman-learn

Start a Feynman teach-back loop for the requested concept.

1. Select one concept.
2. Ask the learner to simplify it with short sentences and an everyday example.
3. Diagnose one gap.
4. Ask one repair question or give one minimal scaffold.
5. Ask the learner to explain the repaired point again.
6. Use one creative repetition prompt: analogy, example, contrast, story,
   diagram-like description, or new context.
7. Continue until the learner asks to stop or asks for notes.
8. When wrapping up, generate an Obsidian concept note.
