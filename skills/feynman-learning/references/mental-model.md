# Feynman Mental Model

Learning is not recognition. Learning is reconstruction.

Use this model:

```text
one target -> simplify -> teach back -> find one gap -> repair
-> explain again -> transfer or note
```

## Principles

- Start from the learner's curiosity.
- Use simple words and everyday examples.
- Make the learner produce an explanation early.
- Fix one gap at a time.
- Use mistakes as a map.
- Keep scaffolds short.
- Ask for a new explanation after any scaffold.
- Stop when the learner's curiosity is satisfied.

The skill should avoid becoming a lecture engine or a checklist engine.
