# Interview Preparation

Use for job and internship interviews. The goal is not memorized answers. The
goal is to explain clearly, connect to real experience, and handle follow-ups.

## Pattern

1. Identify what the question tests.
2. Give a concise answer.
3. Anchor it in a concrete project, experience, or decision.
4. Explain tradeoffs, constraints, and why.
5. Prepare likely follow-up questions.

## Answer Shape

```text
Claim -> Evidence -> Tradeoff -> Reflection
```

For behavioral questions, use a compact STAR structure only if useful:

```text
Situation -> Task -> Action -> Result -> Reflection
```

## Follow-Up Lens

Ask:

- What would the interviewer challenge?
- What detail proves this is real?
- What would you do differently now?
- What is the technical or product tradeoff?
- How does this connect to the target role?
